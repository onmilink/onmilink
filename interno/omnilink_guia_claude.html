1.1

Qué es Claude y cómo funciona

Claude es un modelo de lenguaje grande (LLM) desarrollado por Anthropic. No recupera información de internet en tiempo real (salvo que actives la búsqueda web): genera respuestas basándose en el conocimiento adquirido durante su entrenamiento y en el contexto que tú le proporcionas en cada conversación.

Su funcionamiento es estadístico: predice la continuación más coherente y útil a partir del texto recibido. Por eso el contexto que aportas determina directamente la calidad del resultado. Cuanto más específico seas —cliente, objetivo, datos, tono, formato esperado— menos iteraciones necesitarás.

Su conocimiento tiene corte a principios de agosto de 2025. Para información más reciente (tendencias, precios, noticias de un sector), activa la búsqueda web.

1.3

Tokens, contexto y cuándo preocuparse

¿Qué es un token? Aproximadamente 3–4 caracteres, o entre media y una palabra. "Marketing digital" son 3–4 tokens. Todo el texto que entra (tu mensaje, archivos adjuntos, historial de conversación) y todo el que sale (la respuesta de Claude) consume tokens.

Ventana de contexto: cada conversación tiene un máximo de 200.000 tokens —equivalente a unas 150.000 palabras o ~500 páginas de texto— que Claude puede "ver y recordar" simultáneamente. En la práctica, es prácticamente ilimitado para el trabajo habitual de la agencia.

Límites de uso del plan Team: Claude Team no mide el uso en tokens sino en mensajes por período. Los límites concretos los puedes consultar siempre actualizados en support.claude.com, ya que Anthropic los ajusta periódicamente. Lo relevante: el plan Team ofrece una cuota significativamente mayor que el plan Pro y se reparte por miembro del equipo. Cuando te acerques al límite, Claude te avisará en la interfaz.

1.4

Funcionalidades clave de Claude.ai

📁

Projects

Espacios de trabajo persistentes con contexto fijo. Los documentos y las instrucciones que configures en un Project están disponibles en todos los chats dentro de él, sin tener que repetirlos.

⚡

Skills

Documentos especializados de conocimiento que se añaden al Project para que Claude los use como referencia experta. Ver sección 1.5 para más detalle.

🔍

Búsqueda web

Permite consultar información actualizada en tiempo real: tendencias del sector, precios, noticias de un cliente o benchmarks de campañas. Actívala cuando el contexto temporal sea clave.

🖼️

Análisis de imágenes

Claude ve y analiza capturas de pantalla del Business Manager, gráficas de rendimiento, creatividades, diseños o cualquier imagen que adjuntes. No necesitas describir lo que ves: muéstraselo.

📄

Carga de archivos

Sube PDFs, CSVs, Excels, Word o presentaciones. Claude los lee, analiza y trabaja con ellos dentro de la conversación. Ideal para pasar exports de plataformas directamente.

📑

Artifacts

Genera documentos, tablas o código en un panel lateral independiente del chat. Puedes copiar o descargar el resultado sin seleccionar texto. Perfecto para informes y calendarios.

🧠

Memoria

Claude puede recordar preferencias y contexto de conversaciones anteriores si activas la memoria. Útil para que recuerde el estilo de trabajo preferido de cada miembro del equipo.

👥

Colaboración en equipo

Exclusivo de Claude Team: los Projects pueden compartirse entre todos los miembros. El mismo contexto de cliente, mismos documentos y mismas instrucciones disponibles para todo el equipo desde el primer día.

1.5

Skills: conocimiento especializado para tus Projects

Una Skill es un documento estructurado de conocimiento que subes a un Project para que Claude lo use como referencia experta dentro de ese contexto. No es un documento de texto cualquiera: está redactado con el propósito específico de instruir a Claude sobre cómo comportarse, qué saber y cómo responder en ese Project.

La diferencia entre un documento de referencia normal y una Skill es la intención: un documento informa (datos, histórico, briefing); una Skill instruye (cómo analizar, qué tono usar, qué criterios aplicar, qué formato producir).

🎯

Skill de tono de marca

Instrucciones detalladas sobre el lenguaje, registro y personalidad de un cliente. Claude las aplica automáticamente en todos los copies y comunicaciones del Project.

📊

Skill de análisis de campañas

Define los KPIs prioritarios, umbrales de alarma, estructura del informe y criterios de interpretación de métricas para un cliente concreto.

✉️

Skill de email marketing

Estructura de los flujos de email, segmentos de audiencia, lógica de automatizaciones y plantillas de referencia para cada tipo de envío.

🖌️

Skill de briefing de diseño

Paleta de colores, tipografías permitidas, reglas de uso de logo, formatos estándar de entrega y ejemplos de piezas aprobadas anteriormente.

💡 Cómo crear una Skill efectiva Una Skill no es un PDF de 40 páginas: es un documento conciso (1–3 páginas) con instrucciones directas en segunda persona. Usa frases como "Cuando analices resultados de campaña para este cliente, prioriza siempre ROAS sobre CPA" o "El tono de la marca es directo y sin tecnicismos. Nunca uses palabras como 'sinergia' o 'innovador'". Cuanto más específicas las instrucciones, más consistente será el output de Claude.

1.7

Consejos de uso eficiente

🎯

Contexto > extensión del prompt

No hace falta escribir un texto larguísimo: sí hace falta dar el rol, el objetivo, el cliente y el formato esperado. Esos cuatro elementos evitan el 80 % de las iteraciones.

📊

Datos crudos, no descripciones

En lugar de "las campañas fueron más o menos bien", pega la tabla. Claude analiza números mejor que relatos. Sube el CSV o la captura de pantalla directamente.

🔄

Itera en el mismo chat

"Hazlo más corto", "Añade urgencia", "Versión para TikTok", "Más informal". No empieces de cero: el contexto ya está ahí y la iteración es mucho más rápida.

📌

Pide variantes para testear

Especifica siempre "dame 3 versiones con distintos ángulos" o "una versión emocional y una racional". Las variantes son el input más valioso para los A/B.

🖼️

Muéstrale capturas de pantalla

En lugar de describir cómo está quedando un diseño o un anuncio activo, adjunta la imagen. Claude puede analizar creatividades, gráficas de rendimiento y layouts directamente.

💾

Guarda los prompts que funcionan

Cuando un prompt genere un resultado excelente de forma consistente, guárdalo en una nota del equipo. Construir una librería de prompts por cliente es un activo que crece con el tiempo.

Cómo puede ayudarte en tu trabajo diario

Casos de uso específicos por equipo, con ejemplos de prompts reales y directamente utilizables.

📈

Paid Media

  • Análisis de métricas y diagnóstico de campañas

    Pega los datos de rendimiento (ROAS, CPA, CTR, CPM, frecuencia, conversiones por campaña y conjunto de anuncios) y pide a Claude que identifique patrones, anomalías o palancas de mejora. Puede comparar períodos, audiencias o creatividades y sacar conclusiones con acciones concretas.

  • Redacción de informes ejecutivos para cliente

    Proporciona los datos del mes y el contexto del cliente. Claude estructura el informe con conclusiones primero, datos como soporte y próximos pasos concretos. El resultado es un borrador ejecutivo listo para revisar, no para reescribir. Úsalo como Artifact para descargarlo directamente.

  • Hipótesis y diseño de experimentos A/B

    Describe el problema de rendimiento y el contexto. Claude ayuda a formular la hipótesis, definir la variable de test, la métrica de éxito, el período mínimo necesario y el criterio de victoria. También puede documentar los aprendizajes tras el test.

  • Copies para anuncios con variantes por ángulo

    Especifica producto, audiencia, objetivo (awareness, conversión, retargeting) y plataforma. Claude genera 4–6 variantes con ángulos distintos: beneficio racional, emocional, urgencia, prueba social, miedo a perder (FOMO), precio. Incluye headline, texto principal y CTA adaptados al límite de caracteres de cada plataforma.

  • Estrategia de estructura de campañas y audiencias

    Describe el cliente, objetivo, presupuesto y fase del embudo. Claude propone la arquitectura completa: objetivos de campaña, conjuntos de anuncios, segmentación, distribución de presupuesto y lógica de retargeting.

  • Briefings de cliente y resúmenes de reuniones

    Pega las notas de una call o los puntos discutidos en un WhatsApp. Claude estructura el briefing con objetivo, KPIs, recursos disponibles, plazos y próximos pasos con responsable.

Prompt — Análisis de campaña Meta Ads
"Analiza estos resultados de campañas de Meta Ads para GNG.la durante octubre. Identifica las 3 principales causas de bajo rendimiento y propón acciones concretas para noviembre. Contexto: el objetivo es ventas directas, ROAS mínimo aceptable es 3. Datos: [pegar tabla de campañas con impresiones, clics, gasto, conversiones, ROAS por conjunto de anuncios]."
Prompt — Copies con variantes para Meta Ads
"Genera 5 variantes de anuncio para Meta Ads para el nuevo drop de GNG.la [nombre del producto]. Objetivo: conversión directa. Audiencia: hombres 18-30, intereses streetwear, skateboarding, música urbana. Formato: headline (máx. 40 caracteres) + texto principal (máx. 120 caracteres) + CTA. Ángulos: beneficio de producto, escasez, identidad cultural, precio, prueba social. Tono: directo, sin corporativo, sin clichés de moda."

Social Media

  • Calendarios de contenido mensuales

    Indica cliente, mes, frecuencia, pilares de contenido y fechas clave (drops, campañas, eventos). Claude genera el calendario completo: fecha, formato (reel, carrusel, foto estática, story), tema, copy de referencia y hashtags. Lo puedes pedir como tabla Artifact directamente exportable.

  • Copies para posts, captions y stories

    Claude adapta el tono a cada cliente y plataforma: más cercano para Instagram, más ágil para TikTok, más informativo para LinkedIn. Puede generar varias versiones del mismo post para probar cuál genera más engagement, con y sin emojis, larga y corta.

  • Briefings creativos para reels y vídeos

    A partir de un objetivo o mensaje, Claude estructura el briefing del vídeo: hook de apertura (primeros 3 segundos), desarrollo del contenido, textos en pantalla, música recomendada, CTA final y duración estimada. Punto de partida sólido antes de grabar o editar.

  • Respuestas de community management

    Pega comentarios o mensajes de seguidores y Claude propone respuestas adaptadas al tono de la marca: gestión de quejas, preguntas frecuentes de producto, interacciones positivas o mensajes ambiguos que requieren tacto. Útil cuando hay picos de volumen o situaciones delicadas.

  • Análisis de contenido y benchmarking de competencia

    Describe o comparte capturas de la actividad en redes de un competidor. Claude identifica tendencias, formatos más usados, mensajes recurrentes y oportunidades de diferenciación para el cliente.

  • Adaptación de copies entre formatos y plataformas

    Toma un texto largo (blog, nota de prensa, email) y pide a Claude que lo adapte a publicación de Instagram, story, TikTok de 30 segundos o hilo. Misma información, cada formato con su lógica propia.

Prompt — Calendario mensual Instagram GNG.la
"Crea el calendario de contenidos para Instagram de GNG.la para noviembre. Frecuencia: 4 publicaciones semanales (lunes, miércoles, viernes y domingo) + stories diarias. Pilares: producto (40%), lifestyle/cultura (30%), detrás de cámaras (20%), comunidad/UGC (10%). Fechas clave: lanzamiento drop 'Invierno 24' el día 8, Black Friday el día 29. Formato de salida: tabla con columnas fecha / tipo de formato / tema / copy de referencia / hashtags."
Prompt — Briefing de reel para GNG.la
"Crea el briefing de un reel de 30 segundos para Instagram de GNG.la. Objetivo: mostrar el proceso de fabricación y teñido de la nueva colección para generar expectativa antes del drop del día 8. Incluye: hook de los primeros 3 segundos (sin texto hablado), secuencia de planos sugerida, texto en pantalla en cada momento, tipo de música (referencia de género o artista), y CTA final. Tono: crudo, artesanal, sin filtro estético genérico."

Diseño Gráfico

  • Generación de prompts para imágenes con IA

    Describe la imagen que tienes en mente y Claude la traduce al lenguaje técnico que herramientas como Midjourney, Adobe Firefly o DALL·E entienden mejor: estilo, iluminación, composición, paleta, ratio, referencias de artista, cámara simulada. Ahorra múltiples iteraciones en las herramientas.

  • Briefings de diseño estructurados desde conversaciones con el cliente

    Pega las notas de una reunión, los mensajes de WhatsApp o el email del cliente. Claude extrae y estructura el briefing: objetivo de la pieza, formatos necesarios, paleta y tipografía si se mencionan, referencias visuales, restricciones y plazo. Evita malentendidos y reduce idas y venidas.

  • Textos para piezas gráficas (títulos, claims, copy de carrusel)

    Claude genera los textos que irán dentro de los diseños: titulares, taglines, copy de carrusel diapo a diapo, texto de packaging, claims publicitarios para banner o OOH. Reduce el tiempo de coordinación entre diseño y copy.

  • Moodboard verbal y universo visual de referencia

    Describe el cliente y el proyecto. Claude genera una descripción detallada del universo visual recomendado: paleta de colores con justificación, estilo tipográfico, sensación que debe transmitir, marcas de referencia estética y qué elementos evitar. Input útil antes de empezar un proyecto de branding o rediseño de piezas.

  • Revisión de textos en piezas antes de la entrega

    Antes de enviar un entregable al cliente, pega todos los textos que aparecen en el diseño. Claude revisa ortografía, consistencia de tono, jerarquía de mensajes y si el copy cumple el objetivo de la pieza. Elimina errores que se cuelan cuando estás mirando píxeles y no texto.

  • Memoria de diseño y presentación de propuestas al cliente

    Claude redacta la justificación de las decisiones de diseño para acompañar una propuesta: por qué estos colores, por qué esta tipografía, qué transmite este concepto. Profesionaliza la entrega y facilita la venta del trabajo creativo.

"Necesito un prompt para Midjourney para una foto de producto de GNG.la: hoodie negro mate con bordado tonal en el pecho. Estilo editorial streetwear, no comercial genérico. Fondo urbano desenfocado, luz natural lateral, modelo masculino fuera de encuadre (solo se ve el torso). Sin postproducción excesiva, estética de fotografía analógica. Formato horizontal 16:9. Genera 3 versiones del prompt con variaciones de ambiente: día nublado / noche urbana / interior industrial."

"A partir de las siguientes notas de reunión con el cliente [X], redacta un briefing de diseño estructurado con estas secciones: 1) Descripción del proyecto y objetivo de la pieza, 2) Formatos a entregar con dimensiones, 3) Tono visual y referencias mencionadas, 4) Elementos de marca obligatorios (logo, colores, tipografías si se especifican), 5) Restricciones o elementos que el cliente no quiere, 6) Plazo de entrega. Notas: [pegar notas]."
Transversal

Buenas prácticas para todo el equipo

🔐

Datos sensibles

No incluyas contraseñas, datos bancarios ni información personal identificable. Los datos de campaña (métricas, creatividades, presupuestos) son seguros de compartir. Evita subir contratos completos o documentos legales.

✅

Revisa siempre antes de enviar

Claude puede inventar datos, citar métricas incorrectas o proponer algo que no encaje con la marca. Trátalo como un borrador avanzado que siempre requiere tu criterio profesional antes de llegar al cliente.

📚

Librería de prompts del equipo

Cuando un prompt genere resultados excelentes de forma consistente, guárdalo en un documento compartido del equipo. Esta librería es uno de los activos más valiosos a medio plazo.

🔗

El contexto que das = la calidad que recibes

Claude no tiene acceso a vuestros clientes ni plataformas si no se los proporcionáis. Cada conversación nueva empieza en blanco salvo lo configurado en el Project. Cuanto mejor configurado esté el Project, menos contexto necesitaréis dar en cada chat.
